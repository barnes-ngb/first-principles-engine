# Phase 3 — Records + Compliance Pack
Date: 2026-02-07

## Objective
Make records effortless and exportable:
- Hours totals and breakdowns
- Daily log diary
- Portfolio highlights
- Evaluations
- One-click “pack” export (CSV + Markdown first)

## In-Scope Deliverables
- Hours hardening: derive + adjust + audit trail
- Monthly evaluation workflow (wins/struggles/next steps + sample artifacts)
- Monthly “Demo Night” highlights view (manual select + auto-suggest)
- Export pack:
  - Hours summary CSV
  - Daily log CSV
  - Evaluation markdown
  - Portfolio index markdown (artifact links)

## Acceptance Criteria
- Export pack for any date range with consistent formatting.
- Evaluations are quick enough to do monthly.
